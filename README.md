# Conversor-de-moneda
Proyecto de java que consume una api con los valores actuales de divisas de diferentes paises para poder convertir el equivalente de una divisa a otra

<h1>Características:</h1>

Conversión entre dólares, pesos argentinos, reales brasileños y pesos colombianos

Manejo básico de errores en opciones inválidas

Consumo de API REST con HttpClient

Uso de Java 17, enum para monedas y JSON con Gson

<h1>Requisitos</h1>

Java 17 o superior

Conexión a internet para acceder a la API

API Key válida de ExchangeRate-API

<h1>Como usar:</h1>

Clona o descarga el repositorio

Abre el proyecto en IntelliJ (u otro IDE)

Configura tu API key en ConversorService.java (constante API_KEY)

Ejecuta la clase Principal.java

Sigue las instrucciones en consola para elegir conversión y cantidad
